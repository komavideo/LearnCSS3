外置CSS样式
===========

## 知识点

* 将CSS样式提取到外部的样式文件

## 实战演习

### index.html

~~~html
<!DOCTYPE html>
<html lang="zh-cmn-Hans">
<head>
    <meta charset="utf-8" />
    <title>CSS3入门与理解 | 小马技术</title>
    <link rel="stylesheet" type="text/css" href="style.css" />
</head>
<body>
    <h1>CSS3入门与理解</h1>
    <p>东南西北东南西北东南西北东南西北东南西北东南西北东南西北东南西北东南西北</p>
</body>
</html>
~~~

### style.css

~~~css
h1 {
    color: red;
}
p {
    color: blue;
}
~~~
